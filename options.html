How It Works

This extension activates when a webpage includes the response header:

X-AI-Assist: on

Optional configuration can be provided via:

  • Response Header: X-AI-Config: <base64url-encoded-json>
  • Page Script: <script type="application/json" id="ai-config">{...}</script>

The extension will collect context from elements marked with data-ai-context attributes and automatically fill form fields based on AI-generated content.
